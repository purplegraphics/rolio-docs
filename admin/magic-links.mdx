## What are Magic Links?

A magic link is a personalized, one-time URL sent to a member's email. When clicked, it opens a pre-filled form where they can update their profile — no login or password required.

## Sending a Magic Link

### To a Single Member

1. Go to the member's profile in the Directory
2. Click **Send update request**
3. Choose whether to **auto-approve** updates
4. Click **Send Link**

### Bulk Send

1. Go to **Admin → Magic Links**
2. Select **All members** or a specific group
3. Toggle **Auto-approve updates** if desired
4. Click **Send Update Requests**

<Warning>
  Bulk sends are capped at 200 members per request.
</Warning>

## The Approval Queue

When a member submits an update, it appears in **Admin → Magic Links → Pending Updates**.

Each update shows:

- Member name and email
- When it was submitted
- A diff of what changed (old value → new value in green)
- Approve and Reject buttons

## Auto-Approve

Toggling **Auto-approve updates** applies changes immediately without admin review. Use for trusted members or annual directory updates.

## Magic Link Expiry

- Update links expire after **7 days**
- Login links expire after **24 hours**
- Expired links show a clear message directing members to request a new one
