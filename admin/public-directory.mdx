## Your Public Directory URL

Every Rolio organization gets a public directory at:
`https://www.getrolio.app/view/[your-slug]`

You can find and share this link from **Admin → Overview**.

## Visibility Settings

Control what different audiences can see in **Admin → Settings → Public Directory**.

See [Customize Your Directory](/getting-started/customize-directory) for full details on each setting.

## Member Login

Members can access the directory by verifying their email:

1. Member visits your public directory URL
2. Clicks **Member? Sign in**
3. Enters their email address
4. Receives a one-time access link
5. Clicks the link → authenticated for 30 days

<Note>
Member login only works if the member's email address exists in your directory. This prevents unauthorized access.
</Note>

## Embed Widget

Add your directory to your existing website with an iframe:

```html
<iframe 
  src="https://www.getrolio.app/embed/your-slug" 
  width="100%" 
  height="600" 
  frameborder="0"
  title="Your Org Member Directory">
</iframe>
```

Copy this code from **Admin → Settings → Embed Directory**.

The embed respects all your public visibility settings.
