## Directory URL

Your directory is available at: `https://getrolio.app/view/[your-slug]`

To share it:
- Copy the link from **Admin → Overview**
- Download the QR code for print materials
- Embed it on your website (see Embed Widget below)

## Public Visibility Settings

Control what visitors see without logging in. Go to **Admin → Settings → Public Directory**:

| Setting | Default | Description |
|---|---|---|
| Require login to view | Off | When on, only authenticated members can view |
| Show member bio | On | Display member's about text |
| Show social links | On | Display social media links |
| Show groups | On | Display group memberships |
| Show birthday | Off | Show month and day (never birth year) |
| Show phone number | Off | ⚠️ Enable only with member consent |
| Show email address | Off | ⚠️ Enable only with member consent |
| Show address | Off | ⚠️ Carefully consider privacy |

## Embed Widget

Add your directory to your existing website:

1. Go to **Admin → Settings → Embed Directory**
2. Copy the iframe code
3. Paste it into your website's HTML

The embed respects all your visibility settings.

## Email Settings

Customize how emails appear to your members:

Go to **Admin → Settings → Email Settings**:
- **From name** — How your org appears in email (default: "[Org Name] via Rolio")
- **Notify on member add** — Send welcome email when a member is added (default: off)
- **Notify on member remove** — Send notification when a member is removed (default: off)
