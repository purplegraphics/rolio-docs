## Admin Seat Limits

| Plan | Admin Seats |
|---|---|
| Trial | 1 |
| Starter | 1 |
| Pro | 3 |
| Organization | Unlimited |

## Inviting an Admin

1. Go to **Admin → Settings → Team**
2. Click **+ Invite admin**
3. Enter the email address and select a role:
   - **Admin** — Full access (members, settings, billing)
   - **Manager** — Can manage members and send magic links, but not settings or billing
4. Click **Send Invitation**

The invitee receives an email with an invitation link. If they don't have a Rolio account, they'll be prompted to create one.

<Note>
Pending invitations count toward your admin seat limit.
</Note>

## Removing an Admin

From **Admin → Settings → Team**, click the three-dot menu next to an admin and select **Remove**.

You cannot remove yourself if you are the only admin.

## Promoting a Member to Admin

If an existing member needs admin access:

1. Go to **Admin → Members**
2. Click the three-dot menu next to the member
3. Select **Promote to admin**

<Warning>
The member must already have a Rolio account with the same email address. If they don't, invite them as an admin instead.
</Warning>
