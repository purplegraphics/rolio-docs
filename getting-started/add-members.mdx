---

## Option 1: Add Manually

1. Go to **Directory** in the sidebar
2. Click **+ Add member**
3. Fill in the member's basic information (name, email, phone)
4. Click **Save**

You can add more details (address, birthday, bio, social links) from the member's profile page.

## Option 2: Import via CSV

For bulk imports:

1. Go to **Admin → Members**
2. Click **Import CSV**
3. Upload your file (CSV, Excel, or vCard)
4. Rolio's AI maps your columns to the correct fields automatically
5. Review the preview and confirm the import

<Tip>
Rolio detects and flags duplicate records during import. Review them before confirming.
</Tip>

**Supported import formats:**
- CSV (.csv)
- Excel (.xlsx, .xls)
- vCard (.vcf)

## Option 3: Magic Link Self-Entry

For new members who don't have existing records:

1. Create a basic record with just their name and email
2. Send them a magic link update request
3. They fill in their own details from any device

## Member Fields

Standard fields available for all members:

| Field | Description |
|---|---|
| First & last name | Required |
| Email | Used for magic links and communications |
| Phone | Mobile, home, or work |
| Address | Street, city, state, zip |
| Profile photo | Circular avatar (JPG, PNG, WebP) |
| Cover photo | Banner image (1400×400px) |
| Birthday | Month/day (year optional) |
| Bio | Short about text (max 500 characters) |
| Social links | Facebook, Instagram, Twitter/X, LinkedIn, Website |
| Groups | Which groups the member belongs to |
